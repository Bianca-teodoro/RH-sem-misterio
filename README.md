# RH-sem-misterio
Este projeto começou com a elaboração de um fluxograma e primeiro rascunho em pseudocódigo. A partir destes, foi elaborada a primeira versão do algoritmo em VS Code. Então, o código foi aprimorado para incluir ‘if’s de segurança. Testes foram realizados e foi elaborada uma apresentação de slides em Power Point, que serviu de base para o pitch  do projeto em vídeo.
